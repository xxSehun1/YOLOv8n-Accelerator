# Codex Git Guide

- Commit messages must include `xxsehun`.
- Work directly on `main`.
- Push directly to `origin main`.
- Do not create or push feature branches unless the user explicitly asks for one.
- Before committing, run `git status --short` and review the changed files.
- Use this commit form by default:

```bash
git add <files>
git commit -m "xxsehun: <short description>"
git push origin main
```
